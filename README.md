# Symbol-Wizard

Browser-based symbol RPG prototype with modular architecture and a design-first documentation set.

## Run
Use any static server, for example:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Controls
- Move: `W A S D`
- Aim: Mouse
- Cast Magic Bolt: Left click
- Interact with NPC: `E`
- Dialogue choices: `1-9`
